================================================================================
ğŸ› TROUBLESHOOTING
================================================================================

MASALAH: Website tidak muncul
SOLUSI:
- Pastikan membuka index.html
- Cek browser developer console (F12)
- Lihat error messages
- Refresh page (Ctrl+R)

MASALAH: Quiz tidak berfungsi
SOLUSI:
- Tunggu JSON file selesai dimuat
- Cek Network tab di DevTools
- Refresh halaman
- Coba browser lain

MASALAH: Data tidak tersimpan
SOLUSI:
- Jangan gunakan Private/Incognito mode
- Cek browser localStorage settings
- Clear cache dan coba lagi

MASALAH: Tampilan berantakan
SOLUSI:
- Hard refresh (Ctrl+Shift+R)
- Bersihkan browser cache
- Coba browser lain
- Ukuran window terlalu kecil?
